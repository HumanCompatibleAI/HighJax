# Octane Navigation and Playback Quick Guide

Related: [[Octane navigation and playback]] | [[Octane key bindings]] | [[Octane overview]]

## Focus

- **Tab / Shift-Tab**: Cycle panes (Treks -> Parquets -> Epochs -> Episodes -> Highway)
- **t / a / o / e / s**: Jump directly to a pane

## Navigation

- **Up/Down**: Select item in list panes (Treks, Parquets, Epochs, Episodes)
- **Left/Right**: Step frames in Highway
- **j/k**: 1 frame back/forward (works from any pane)
- **J/K**: 5 frames back/forward
- **h/l**: First/last frame
- **Home/End**: First/last item in any pane
- **PageUp/Down**: Jump 80% of visible rows

## Playback

- **p**: Play/pause
- Default speed is 2.0x, adjustable 0.25x-10.0x in Graphics modal (r)
- Auto-stops at episode end

## Jump modals

- **g**: Jump to epoch number
- **G**: Jump to episode index
- **/**: Jump to timestep index
- Type digits, Enter to confirm, **n** to repeat last jump, Esc to cancel

## Memory

Frame positions are remembered per (epoch, episode). Trek selections are remembered when switching between treks. Press **R** to refresh trek data from disk.
