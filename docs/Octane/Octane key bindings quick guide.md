# Octane Key Bindings Quick Guide

Related: [[Octane key bindings]] | [[Octane overview]] | [[Octane navigation and playback]]

## Essential keys

| Key | Action |
|---|---|
| q / Esc | Quit |
| Tab / Shift-Tab | Cycle pane focus |
| t / a / o / e / s | Jump to pane |
| Up / Down | Navigate lists |
| j / k | Frame back/forward |
| J / K | 5 frames back/forward |
| h / l | First/last frame |
| p | Play/pause |

## Modals

| Key | Opens |
|---|---|
| ? | Help |
| r | Graphics settings |
| g | Jump to epoch |
| G | Jump to episode |
| / | Jump to timestep |
| b | Behavior scenario capture |
| B | Behavior Explorer |

## In jump modals

Type a number, Enter to confirm, **n** to repeat last jump, Esc to cancel.

## In Graphics modal

j/k or Up/Down to navigate controls, h/l or Left/Right to adjust. Mnemonic keys: f(fps), s(speed), z(zoom), w(width), p(podium), c(scala), x(sextants), o(octants), v(velocity arrows), i(action dist), t(action dist text), a(attention), n(NPC text), d(debug eye), b(blend mode), e(theme).

## Utility

| Key | Action |
|---|---|
| c | Copy last toast text |
| d | Toggle debug overlay |
| C | Copy octane command |
| D | Copy debug info |
| y | Dump frame state to JSON |
| R | Refresh trek from disk |
| Ctrl+L | Copy log file path |
| W | Sidebar width (opens Graphics) |
